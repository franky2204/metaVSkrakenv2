1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/metaVSkrakenv2.git
    ```
2. Navigate to the project directory:
    ```bash
    cd metaVSkrakenv2
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
